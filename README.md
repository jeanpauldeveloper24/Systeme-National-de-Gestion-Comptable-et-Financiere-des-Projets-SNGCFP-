# Systeme-National-de-Gestion-Comptable-et-Financiere-des-Projets-SNGCFP-
Ce dépôt est l’espace officiel du Système National de Gestion Comptable et Financière des Projets (SNGCFP) en Côte d’Ivoire, financé par la BAD. Il centralise le code source, les configurations et la documentation pour une plateforme sécurisée, conforme aux normes institutionnelles et internationales de gestion des finances publiques.
